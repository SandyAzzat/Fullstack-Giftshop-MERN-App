# FYONKA E-Commerce

**Fyonka** is a full-stack **gift shop e-commerce website** built with the **MERN stack** (MongoDB, Express, React, Node.js).  
It allows users to easily browse, choose, and purchase gifts for their loved ones.

This project was developed as the **Final Graduation Project** for the **AMIT & EFE Programs**.

## Features
- Browse and search for unique gifts  
- Add products to cart or wishlist  
- Secure checkout and payment process  
- User authentication and profile management  
- Admin dashboard for managing products, orders, and users  
- Responsive and modern UI design  

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Token)
